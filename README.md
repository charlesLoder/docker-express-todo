# ToDo-app-node-js-express-js-mongodb

## prereqs

- have [Docker installed](https://docs.docker.com/engine/install/)

## quick start

Run

```bash
git clone git@github.com:classicalacademicpress/docker-todo-app.git
cd docker-todo-app
npm install
docker compose up
```

Navigate to http://localhost:3000
